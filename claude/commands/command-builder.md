⚡ クイックコマンド作成 - 30秒で新しいコマンドを作成

引数: $ARGUMENTS

使い方: /quick-cmd "コマンドの説明を日本語で"

例:
- /quick-cmd "全てのTypeScriptファイルをフォーマットする"
- /quick-cmd "未使用のnpmパッケージを検出して削除"
- /quick-cmd "今日の作業内容をまとめてレポート作成"

実行内容:
1. 説明文「$ARGUMENTS」を解析
2. 適切なコマンド名を生成（kebab-case）
3. 必要な処理を特定
4. コマンドファイルを生成して保存
5. 「/[生成されたコマンド名]」で実行可能に

30秒以内に使えるようになります！
